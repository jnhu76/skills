# Skill Name Migration

This repository now uses short personal skill keys with the `jn_` prefix.

## Mapping

| Old name | New name |
|---|---|
| `project-boundary-auditor` | `jn_audit` |
| `bug-or-boundary` | `jn_bug` |
| `company-generation-screener` | `jn_company` |
| `sandbox-gap-finder` | `jn_sandbox` |
| `semantic-extraction` | `jn_semantic` |
| `english-vocab-card-builder` | `jn_vocab` |
| `english-sentence-pattern-miner` | `jn_sentence` |
| `english-listening-chunk-trainer` | `jn_chunk` |
| `english-output-feedback-coach` | `jn_feedback` |

## Rationale

The old names were descriptive but too long to remember during daily use.
The new names use a consistent `jn_` prefix and short memorable keys.

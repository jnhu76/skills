# Skill Name Migration

This repository now uses short personal skill keys with the `jn-` prefix.

## Mapping

| Old name | New name |
|---|---|
| `project-boundary-auditor` | `jn-audit` |
| `bug-or-boundary` | `jn-bug` |
| `company-generation-screener` | `jn-company` |
| `sandbox-gap-finder` | `jn-sandbox` |
| `semantic-extraction` | `jn-semantic` |
| `english-vocab-card-builder` | `jn-vocab` |
| `english-sentence-pattern-miner` | `jn-sentence` |
| `english-listening-chunk-trainer` | `jn-chunk` |
| `english-output-feedback-coach` | `jn-feedback` |

## Rationale

The old names were descriptive but too long to remember during daily use.
The new names use a consistent `jn-` prefix and short memorable keys.

# Agent Skill Hub

个人 Agent Skill 仓库。

## Skills

| Skill | 用途 |
|---|---|
| [`bounded-review-context`](skills/bounded-review-context/) | 请求代码审查前生成 scoped Review Context Pack，限制上下文收集并沿当前 diff impact chain 指导 reviewer |
| [`construction-kickoff-supervisor`](skills/construction-kickoff-supervisor/) | 实现前把 Job Card / plan 转成 TDD-ready Job Pack，明确 Scope Lock、Context7 证据、TDD slices 和停工条件 |
| [`jn-audit`](skills/jn-audit/) | 分析软件项目真实边界，判断 bug / 边界 / 功能缺口 / 技术债 |
| [`jn-bug`](skills/jn-bug/) | 快速判断一个具体现象是 bug、边界、误用、环境问题还是功能缺口 |
| [`jn-company`](skills/jn-company/) | 通过中国产业代际框架分析公司，判断属于第6/7/8代，区分真实投资标的与叙事股 |
| [`jn-sandbox`](skills/jn-sandbox/) | 分析 AI Agent 沙盒系统，从第一性原则找到研发或科研缝隙 |
| [`jn-semantic`](skills/jn-semantic/) | 从技术问题、论文、API、运行时等提取语义模型，将功能堆转换为可操作语义 |
| [`jn-vocab`](skills/jn-vocab/) | 从英文文章和生词列表生成可复习词卡、搭配、例句和 Anki TSV |
| [`jn-sentence`](skills/jn-sentence/) | 从英文文章提炼可迁移句型，用于作文、看图写话和复述 |
| [`jn-chunk`](skills/jn-chunk/) | 分析听力稿 chunk、弱读、连读、重音和 shadowing 跟读 |
| [`jn-feedback`](skills/jn-feedback/) | 批改基于今日生词、词块、句型写出的作文、看图写话或复述稿 |
| [`pharma30days`](skills/pharma30days/) | 近 N 天医药/生物科技新闻研究：按公司、药品、靶点、适应症动态检索、去重聚类、映射管线并判断临床/监管/BD/商业/股价催化 |
| [`writing-coach`](skills/writing-coach/) | 系统训练用户从"事实清单/大纲"升级为"有判断、有比较、有边界、有细节"的分析型写作 |

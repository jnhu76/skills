# Agent Skill Hub

个人 Agent Skill 仓库。

## Skills

| Skill | 用途 |
|---|---|
| [`project-boundary-auditor`](skills/project-boundary-auditor/) | 分析软件项目真实边界，判断 bug / 边界 / 功能缺口 / 技术债 |
| [`bug-or-boundary`](skills/bug-or-boundary/) | 快速判断一个具体现象是 bug、边界、误用、环境问题还是功能缺口 |
| [`company-generation-screener`](skills/company-generation-screener/) | 通过中国产业代际框架分析公司，判断属于第6/7/8代，区分真实投资标的与叙事股 |
| [`sandbox-gap-finder`](skills/sandbox-gap-finder/) | 分析AI Agent沙盒系统，从第一性原则找到研发或科研缝隙 |
| [`semantic-extraction`](skills/semantic-extraction/) | 从技术问题、论文、API、运行时等提取语义模型，将功能堆转换为可操作语义 |
| [`english-vocab-card-builder`](skills/english-vocab-card-builder/) | 从英文文章和生词列表生成可复习词卡、搭配、例句和 Anki TSV |
| [`english-sentence-pattern-miner`](skills/english-sentence-pattern-miner/) | 从英文文章提炼可迁移句型，用于作文、看图写话和复述 |
| [`english-listening-chunk-trainer`](skills/english-listening-chunk-trainer/) | 分析听力稿 chunk、弱读、连读、重音和 shadowing 跟读 |
| [`english-output-feedback-coach`](skills/english-output-feedback-coach/) | 批改基于今日生词、词块、句型写出的作文、看图写话或复述稿 |

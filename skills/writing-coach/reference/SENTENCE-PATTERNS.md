# Sentence Patterns

## Core judgment

```text
A 不只是 ______ ，而是 ______ 。
```

```text
这件事的关键不在于 ______ ，而在于 ______ 。
```

```text
过去我们以为 ______ ，但真正发生变化的是 ______ 。
```

## Mechanism

```text
它之所以有效，是因为 ______ 提供了 ______ 的反馈。
```

```text
现实里，它表现为：______ 、______ 、______ 被串成一个循环。
```

```text
这不是凭空创造，而是把 ______ 变成可重复执行的流程。
```

## Cost-structure analysis

```text
过去 ______ 的成本高，不是因为 ______ ，而是因为 ______ 。
```

```text
一旦 ______ 成本下降，______ 就会从稀缺能力变成基础能力。
```

```text
竞争焦点会从“能不能做”转向“谁做得更快、更稳、更便宜”。
```

## Boundary

```text
但这并不意味着 ______ 。真正的瓶颈仍然在 ______ 。
```

```text
这个判断成立的前提是 ______ 。
```

```text
AI 降低的是 ______ 成本，不是直接解决 ______ 问题。
```

## Personal punchline setup

```text
说白了，______ 。
```

```text
前面这些机制合在一起，结果就很现实：______ 。
```

```text
所以不是没人想做，而是以前进不去、试不起、撑不住；现在门槛一降，______ 。
```

# Lesson 0001: Judgment Sentence

## Skill

Turn a raw thought into one clear judgment sentence.

## Why this matters

A technology-analysis article needs a controlling judgment. Without it, the writing becomes a pile of observations.

## Weak example

> AI Agent 很厉害，会改变很多行业。

Problem: too general. It says nothing about what changes.

## Stronger example

> AI Agent 的意义不只是替代程序员写代码，而是降低进入新领域的迁移成本和试错成本。

Why stronger:

- It has contrast: 不只是 A，而是 B。
- It names the real change: 迁移成本、试错成本。
- It can be expanded into a logic chain.

## Exercise

Write 3 judgment sentences using this pattern:

```text
______ 不只是 ______ ，而是 ______ 。
```

Use one of these topics:

- AI Agent
- RISC-V
- 软件迁移
- 开源芯片
- 技术蓝海变红海

## Feedback criteria

- Does the sentence contain a real judgment?
- Does it avoid empty big words?
- Can it be expanded into a 5-step logic chain?

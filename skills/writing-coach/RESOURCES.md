# Resources

## Skill design references

- Matt Pocock, `writing-great-skills/SKILL.md`: reference for predictable, maintainable skills. Key ideas to preserve in this workspace: keep a skill process-driven, separate steps from reference, use progressive disclosure, avoid duplication, and keep one source of truth.
  - URL: https://github.com/mattpocock/skills/blob/main/skills/productivity/writing-great-skills/SKILL.md

- Matt Pocock, `teach/SKILL.md`: reference for a stateful teaching workspace. Key ideas to preserve: mission-driven teaching, learning records, zone of proximal development, short lessons, and durable skill acquisition through retrieval, spacing, and interleaving.
  - URL: https://github.com/mattpocock/skills/blob/main/skills/productivity/teach/SKILL.md

- Matt Pocock, `teach/MISSION-FORMAT.md`: reference for short, concrete mission files.
  - URL: https://github.com/mattpocock/skills/blob/main/skills/productivity/teach/MISSION-FORMAT.md

- Matt Pocock, `teach/LEARNING-RECORD-FORMAT.md`: reference for learning records as non-obvious, decision-grade learning notes, not routine logs.
  - URL: https://github.com/mattpocock/skills/blob/main/skills/productivity/teach/LEARNING-RECORD-FORMAT.md

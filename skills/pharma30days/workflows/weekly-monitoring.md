# Workflow: Weekly / 30-Day Monitoring

## Input

- topic: company / drug / target / indication
- days: default 30, weekly 7
- geography: optional

## Steps

1. Normalize the topic and aliases.
2. Build a search plan from source packs.
3. Search official and English sources first.
4. Search Tencent News / Chinese discovery sources for additional leads.
5. Merge and deduplicate results.
6. Cluster by event.
7. Build or update pipeline registry.
8. Score catalysts.
9. Produce weekly report.
10. Output registry update block.

## Required sections

- Executive summary
- Major clusters
- Asset-stage map
- Catalyst score table
- What changed
- What did not change
- Short / medium / long stock impact
- Next 4-12 week watchpoints
- Anti-hype notes

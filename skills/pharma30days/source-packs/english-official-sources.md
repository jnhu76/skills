# English Official Source Pack

Use these sources first when validating pharma claims.

## Regulatory

- FDA: approval letters, labels, advisory committee docs, drug safety communications
- EMA: EPAR, CHMP opinions, medicine overview
- NMPA / CDE: acceptance, review, priority review, breakthrough therapy, approval status
- ClinicalTrials.gov: trial phase, population, endpoints, sponsor, estimated completion
- EU Clinical Trials Register

## Conferences and publications

- ASCO / ASCOPubs
- ESMO / Annals of Oncology
- AACR / Cancer Discovery
- WCLC / IASLC
- SABCS
- EHA
- JCO
- NEJM
- Lancet / Lancet Oncology
- Nature Medicine

## Company official

- English press release
- investor presentation
- annual report
- interim report
- exchange filing
- SEC / HKEX / SSE / SZSE filing

# English Industry and Financial Media Source Pack

Use these sources after official sources. They are useful for context, competitor tracking, and BD / commercial analysis.

## Core pharma / biotech media

- Reuters
- Bloomberg
- Fierce Biotech
- Fierce Pharma
- BioPharma Dive
- Endpoints News
- BioCentury
- Evaluate Vantage
- STAT
- PharmaTimes
- Scrip / Pink Sheet, if available

## Oncology-focused media

- OncLive
- UroToday
- Targeted Oncology
- Cancer Network
- Oncology News Central
- ESMO Daily Reporter
- ASCO Daily News

## Search query templates

```text
{company English} {drug} Reuters last 30 days
{drug} Fierce Biotech
{drug} BioPharma Dive
{target} ADC competitor approval
{indication} {target} Phase III competitor
{company English} licensing deal upfront milestone
```

## Interpretation rules

- Reuters / Bloomberg can be strong for deals and financial market reaction.
- Industry media are not enough to prove clinical truth if they cite no primary data.
- Oncology media can summarize conference data well, but still check the abstract or poster when possible.

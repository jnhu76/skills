# English Listening Chunk Trainer

## 这个 skill 是干什么的？

把英文听力稿拆成真实语速下的 chunk，并解释为什么一些词会“粘在一起”。

重点训练：

- chunk 意群；
- 弱读；
- 连读；
- 失爆；
- 省音；
- 重音；
- 节奏；
- shadowing 跟读。

## 什么时候用？

适合这些场景：

```text
我有一段听力稿，但真实音频里听不清。
请帮我拆 chunk，分析弱读、连读、重音和跟读方法。
```

## 调用 Prompt

```markdown
请使用 `english-listening-chunk-trainer` skill 分析下面听力稿。

我的目标不是只看懂文本，而是听懂自然语速下这些词如何连在一起。

听力稿：

<粘贴听力稿>

我听不清的地方：

<可选：粘贴听不清的句子或单词>

请输出：

1. 全文意思；
2. chunk 切分；
3. 每个 chunk 的功能；
4. 弱读、连读、失爆、省音、同化；
5. 重音和节奏；
6. 慢速跟读版；
7. 自然语速跟读版；
8. shadowing 跟读版；
9. 听写训练；
10. 今日听力复习清单。
```

# English Vocab Card Builder

## 这个 skill 是干什么的？

把英文文章里的生词整理成可复习、可迁移使用的词卡。

它不是简单查词典，而是帮助学习者完成：

- 原文词形 → 单词原型；
- 本文含义 → 常见含义；
- 变形、搭配、例句；
- 中译英练习；
- Anki / 生词本 TSV 导出。

## 什么时候用？

适合这些场景：

```text
我读了一篇英文文章，有些单词不会。
请帮我整理生词、原型、含义、变形、搭配和例句。
最好能导出 Anki / TSV。
```

## 调用 Prompt

```markdown
请使用 `jn_vocab` skill 整理下面文章中的生词。

我的目标不是只知道中文意思，而是：

1. 还原单词原型；
2. 理解它在本文中的含义；
3. 掌握常见变形；
4. 掌握常见搭配；
5. 能用它造句；
6. 能导入 Anki 或生词本复习。

文章：

<粘贴英文文章>

不会的单词：

<粘贴单词列表>

请输出：

- 生词详解；
- 生词总表；
- 常见搭配；
- 例句；
- 中译英练习；
- Anki TSV。
```

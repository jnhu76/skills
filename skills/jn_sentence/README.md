# English Sentence Pattern Miner

## 这个 skill 是干什么的？

从英文文章中提炼可以迁移使用的句型。

它的目标不是翻译句子，而是把文章里的好句子变成：

- 写作模板；
- 看图写话模板；
- 口头复述模板；
- 中译英训练材料。

## 什么时候用？

适合这些场景：

```text
我看懂了文章，但想把里面的句型学会。
请帮我提炼可复用句型，并给仿写例句。
```

## 调用 Prompt

```markdown
请使用 `jn_sentence` skill 分析下面文章。

我的目标不是只翻译文章，而是提炼以后能在作文、看图写话、复述中使用的句型。

文章：

<粘贴英文文章>

请输出：

1. 文章最值得学习的句子；
2. 每个句子的主干；
3. 句型公式；
4. 适用场景；
5. 仿写例句；
6. 中译英练习；
7. 今日必背句型清单。

仿写例句尽量贴近学习、技术、工作、投资和生活场景。
```

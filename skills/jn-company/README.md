# jn-company

## 这个 skill 是干什么的？

用中国产业代际框架分析上市公司或非上市公司，判断它更接近第 6 / 7 / 8 代产业能力，区分真实投资标的和叙事股。

它不是直接给买卖建议，而是帮助判断：

- 公司真实能力是否已经转化为收入、毛利、现金流、定价权或供应链控制力；
- 叙事是否有财务和业务证据支撑；
- 全球化、资源配置、独立知识产权是否真实存在；
- 是否值得进入下一步投资研究。

## 什么时候用？

适合这些问题：

```text
这家公司是第几代产业公司？
它是真科技公司还是叙事股？
它有没有全球化扩张能力？
它有没有独立 IP 和供应链控制力？
它值不值得继续做投资研究？
```

## 调用 Prompt

```markdown
请使用 `jn-company` skill 分析下面这家公司。

公司：

<公司名称 / 股票代码 / 简介>

我关心的问题：

<投资研究问题或产业判断问题>

请重点判断：

1. 它属于第 6 / 7 / 8 代中的哪一类，为什么；
2. 它的核心能力是否已经变成收入、利润、现金流或定价权；
3. 它是否只是 AI、自主可控、全球化、新质生产力等叙事；
4. 它的全球化、资源控制、供应链地位是否真实；
5. 它还缺哪些关键证据；
6. 最后给出是否值得继续研究，而不是直接买卖建议。
```

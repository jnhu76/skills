# English Output Feedback Coach

## 这个 skill 是干什么的？

批改学习者基于今日生词、词块、句型写出来的英文输出。

它负责把输入学习闭环到输出训练：

```text
文章 / 听力输入
  → 生词 + 句型 + chunk
  → 看图写话 / 作文 / 复述
  → 批改
  → 错误回流
```

## 什么时候用？

适合这些场景：

```text
我用今天学的词和句型写了一段英文。
请检查我是否真的用到了今天的材料，并帮我批改。
```

## 调用 Prompt

```markdown
请使用 `english-output-feedback-coach` skill 批改我的英文输出。

今日生词：

<粘贴今日生词>

今日词块：

<粘贴今日词块>

今日句型：

<粘贴今日句型>

我的英文输出：

<粘贴作文 / 看图写话 / 复述稿>

请输出：

1. 完成度检查：是否使用了今日生词、词块、句型；
2. 语法错误；
3. 搭配错误；
4. 句型使用是否自然；
5. 最小修改版；
6. 更自然版本；
7. 错误回流卡片；
8. 下一轮中译英练习。
```
